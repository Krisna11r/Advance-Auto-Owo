# Advance-Auto-Owo
A Epic OwO Bot Selfbot, which can bypass ban by solving any captcha [ web or imagetotext ]

## Features
- [x] Auto Hunt
- [x] Auto Battle
- [x] Auto Pray
- [x] Auto Level Up 
- [x] Auto Daily
